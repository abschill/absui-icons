# absui-icons

> Icon Pack for Absolute UI (React)
## Installation

```bash
npm install --save absui-icons
```

## Usage
```jsx
import {MenuIcon} from 'absui-icons'
const App = () => {
  return <MenuIcon fill="#c0c0c0" size="lg" />
}
```
## Size Options
- sm - 24x24
- md - 48x48 
- lg - 96x96
