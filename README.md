# absui-icons

> Icon Pack for Absolute UI (React)
## Installation

```bash
npm install --save absui-icons
```

## Usage
```jsx
import {CoffeeIcon} from 'absui-icons'
const App = () => {
  return <CoffeeIcon fill="#c0c0c0" size="lg" />
}
```
## Size Options
- sm - 32x32
- md - 64x64 
- lg - 128x128
